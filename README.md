# Dhyana-95-V2 Camera Control

基于 TUCam SDK 的 **Dhyana-95-V2** 科学相机采集与拉曼光谱数据处理软件。

## 功能概要

### 数据采集

| 功能 | 说明 |
|------|------|
| 曝光时间调节 | 默认 1000 ms，单位 ms，范围受相机硬件限制 |
| 温度控制 | TEC 制冷目标温度，默认 -10 °C，自动开启 TEC |
| 风扇控制 | 四档可调 (1–4)，不允许关闭，默认二档 |
| 设备信息 | 显示型号、序列号、固件版本、传感器尺寸、通道数等 |
| 实时遥测 | FPGA / PCBA / 环境温度，风扇转速，每秒刷新 |
| 单帧采集 | 抓取一帧并显示 |
| 连续采集 | 持续抓取帧，实时预览 |
| 停止采集 | 终止当前采集流程 |
| 保存图片 | 支持 TIFF / PNG / JPEG 格式（需相机连接） |

### 数据处理

| 功能 | 说明 |
|------|------|
| 行分组 | 自定义多组行范围（逗号/空格/换行分隔），如 `1-40, 91-130` |
| 列合并 | 按因子 *n* 将相邻列取平均合并（n=1 不变，n=2 每两列合并取平均） |
| arPLS 基线校正 | 非对称重加权惩罚最小二乘基线估计，参数可调 (λ/迭代/容差) |
| 输出模式 | 原始数据 / 校正后 (data − baseline) / 仅基线 |

### 气体浓度分析

| 功能 | 说明 |
|------|------|
| 峰位检测 | 在预设峰位置 ± 搜索窗口内定位最高点 |
| 浓度计算 | 峰高 × 气体系数 / 所有气体总分量 |
| 气体管理 | 自由增删气体，独立设置峰位置、搜索窗口、系数 |
| 未检出判断 | 净峰高低于噪声阈值时自动标记 |
| 实时浓度表 | 显示当前帧各气体峰高与浓度百分比 |
| 浓度趋势图 | 全部叠加 / 单独气体切换，横轴时间/帧序号 |

### 辅助功能

| 功能 | 说明 |
|------|------|
| TIF 载入 | 单张 TIF 加载用于离线测试 |
| 批量测试 | 选择文件夹，自动按序播放全部 TIF，模拟连续采集 |
| 图像光标 | 左键定位 → 十字准线 + 像素值，方向键移动，右键取消 |
| 光谱光标 | 勾选开关后左键定位 → 竖线 + 水平虚线 + 数值标注，←→ 移动 |
| 图表缩放 | 图谱/浓度页均内置 matplotlib 工具栏（缩放/平移/保存） |
| 断开检测 | 每秒轮询连接状态，意外断开自动停止采集并弹窗 |
| 参数校验 | 行分组格式 / 曝光温度范围 / 气体配置空值 均有弹窗提示 |

### UI 界面 (4 页)

1. **采集 / Acquisition** — 图像实时预览 + 设备信息 + 遥测 + 光标
2. **设置 / Settings** — 相机参数 + 数据处理 + arPLS + 气体配置表
3. **图谱 / Spectrum** — 处理后光谱折线图，全部叠加/单独显示，基线叠加
4. **浓度 / Concentration** — 当前浓度表 + 浓度趋势图

## 环境要求

- **Windows 10/11 64-bit**
- **Python 3.10+**
- **TUCam SDK** 已安装（Dhyana 系列相机驱动）
- 离线测试模式无需相机

## 安装与运行

```bash
# 1. 安装 uv
pip install uv

# 2. 进入项目目录
cd tucam

# 3. 安装依赖
uv sync

# 4. 运行
uv run tucam-control
```

## 项目结构

```
tucam/
├── pyproject.toml
├── README.md
├── .python-version
├── lib/
│   └── x64/                  # TUCam SDK DLL
└── src/
    └── tucam_control/
        ├── main.py            # 入口，stderr 过滤器
        ├── TUCam.py           # SDK ctypes 封装
        ├── camera.py          # 相机控制器
        ├── data_processor.py  # 行分组 / 列合并 / arPLS
        ├── gas_analyzer.py    # 气体峰位检测 / 浓度计算
        └── ui/
            ├── main_window.py      # 主窗口 (4 标签页)
            ├── acquisition_tab.py  # 图像 + 光标 + 批量测试
            ├── settings_tab.py     # 参数 + 气体配置表
            ├── data_tab.py         # 光谱图 + 光标
            └── concentration_tab.py # 浓度表 + 趋势图
```

## 扩展指南

- **新增相机功能** → `camera.py` 的 `CameraController`
- **新增处理算法** → `data_processor.py` 或新建模块
- **新增气体种类** → 设置页气体配置表直接添加
- **新增 UI 页面** → `ui/` 下创建，`main_window.py` 中注册
- **新增 SDK 调用** → `TUCam.py` 添加 ctypes 绑定
