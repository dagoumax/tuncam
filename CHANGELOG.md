# Changelog

## 2026-05-15

### 工作模式切换
- 设置页新增工作模式选择：HDR / High Gain / Low Gain
- 通过 `TUIDC_IMGMODESELECT` 切换，HG 模式读噪最低 1.6e⁻

### 设备状态监控
- 设置页新增「设备状态」区域：连接状态、设备名称、三路温度
- `app.aboutToQuit` 确保退出时正确释放相机线程

### 架构整理
- 抽取 `_mpl_style.py` 共享模块（CJK 字体、COLORS、WINDOW_SIZE、fix_tick_labels）
- 全面代码审查：修复 2 处 Runtime Error、2 处逻辑 Bug、7 处未使用导入
- 修复 `paintEvent`/`keyPressEvent` 重复定义导致的事件错误

### 拉曼校准集成
- 默认线性校准：O2(572→1558 cm⁻¹)、N2(1015→2333 cm⁻¹)
- 横轴始终显示拉曼位移 ($cm^{-1}$)，纵轴改为"强度 / Intensity"
- 气体配置新增「拉曼位移」列，校准后自动修正像素位置
- 修复校准光标点击位置反算、$cm^{-1}$ 上角标渲染

### 性能优化
- 浓度/图谱页重绘改为周期定时器 (500ms)，批量测试不丢帧
- 隐藏标签页跳过重绘，切回时 `showEvent` 自动刷新

### 浓度页面修复
- 修复全部行组模式下 CSV 导出为空、xlim 计算引用错误键
- 修复 `if/elif/else` 逻辑导致 `is_datetime` 被覆盖
- 缩放状态仅在新数据到达时重置，保持用户手动缩放/平移
- 导出默认文件名格式：`concentration_YYYY-MM-DD-HH-MM.csv`
- 修复中文逗号无法解析行分组、`MouseButton` 未导入、`_WINDOW_SIZE` 未定义

## 2026-05-14

### 浓度页面 — 多行组独立显示
- 支持全部行组对比：选择单一气体后切换到「全部行组」，叠加各组的浓度曲线
- 行组选择器新增「全部行组」选项，浓度历史按行组分别存储

### 浓度页面优化
- 横轴改为系统时间 `HH:MM:SS` 格式，批量模式为帧序号
- 100 点滑动窗口，拖拽可查看历史；CSV 导出（UTF-8 BOM，含完整时间戳）

### 连续采集时序修正
- `wait_for_frame` 超时改为 `曝光时间 + 500ms`
- 采集轮询间隔 50ms → 200ms，降低 CPU 占用

### 性能与 Bug 修复
- 隐藏标签页不更新、光标与缩放冲突、标注位置、legend 警告
- 列合并后气体峰位置缩放、libpng 警告、空区间保护等

### 新增功能
- 光谱十字准线光标、批量 TIF 测试、缩放工具栏
- 气体配置表可增删编辑、arPLS 默认开启
