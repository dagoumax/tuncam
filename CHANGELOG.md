# Changelog

## 2026-05-14

### 浓度页面优化
- 横轴改为系统时间 `HH:MM:SS` 格式（相机模式），批量模式仍为帧序号
- 新增 100 点滑动窗口：仅显示最新数据，拖拽可查看历史
- 新增 CSV 导出功能（含完整时间戳，UTF-8 BOM）

### 连续采集时序修正
- `wait_for_frame` 超时改为 `曝光时间 + 500ms`，确保不丢帧
- 采集轮询间隔从 50ms 改为 200ms，降低 CPU 占用
- 确保浓度数据点间隔与曝光时间一致（1000ms 曝光 → ~60 点/分钟）

### 性能与 Bug 修复
- 修复隐藏标签页不更新的问题（增加 `_dirty` 标记 + `showEvent` 重绘）
- 修复光谱光标与缩放工具栏冲突（增加「光标」开关复选框）
- 修复光谱光标标注在放大后消失的问题（改用 axes fraction 坐标）
- 修复 legend 无数据时 `UserWarning` 弹窗
- 修复列合并后气体峰位置越界（`GasAnalyzer` 自动按 `merge_factor` 缩放）
- 修复 `libpng iCCP sRGB profile` 警告（stderr 过滤器）
- 修复 `GasConfig.default_gases` 调用错误
- 修复 `ValueError: attempt to get argmax of an empty sequence`（空区间保护）

### 新增功能
- 光谱页面：十字准线光标（竖线 + 水平虚线交汇）
- 批量 TIF 测试：选择文件夹自动循环播放，模拟连续采集
- 图谱/浓度页均内置 matplotlib 缩放工具栏
- 浓度表移除「分量」列，精简为「气体 | 峰高 | 浓度」
- 设置页新增气体配置表：可添加/删除/编辑气体参数
- 光标：左键定位，方向键微调，右键取消

### 默认设置变更
- arPLS 基线校正默认开启
- 输出模式默认「校正后」(corrected)

## 2026-05-13 及之前

### 初始版本
- TUCam SDK 封装 (ctypes)，DLL 自动路径定位
- 相机控制：曝光/温度/风扇/设备信息/单帧/连续采集
- 数据处理：行分组、列合并取平均、arPLS 基线校正
- 气体分析：峰位检测、浓度计算、可配置气体库
- UI：4 页标签 (采集/设置/图谱/浓度)
- TIF 载入离线测试
- 设备断开检测与弹窗告警
- 参数校验与格式化错误提示
